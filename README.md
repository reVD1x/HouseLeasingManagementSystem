房屋租赁管理系统
项目基本功能模块：
o 房源管理模块：房东或管理员录入房源信息，包括房屋地址、户型、面积、租金、装修情况、配套设施等，可对房源进行查询、编辑、删除和推荐操作。
o 租客管理模块：记录租客信息，如姓名、身份证号、联系方式、租赁期限等，可对租客进行查询和管理。
o 租赁合同管理模块：生成租赁合同模板，在线签订租赁合同，记录合同起止时间、租金支付方式、违约条款等，并可对合同进行查询、打印和存档。
o 租金管理模块：设置租金收取标准和时间，记录租金支付情况，生成租金报表，提醒租客缴纳租金，对逾期未付租金进行处理。
o 维修管理模块：租客提交维修申请，管理员或房东安排维修人员进行维修，记录维修情况和费用。
o 评价管理模块：租客和房东可以对租赁过程进行评价和反馈，提高服务质量。
o 统计分析模块：对房源出租情况、租金收入、租客信息等进行统计分析，为决策提供数据支持。
o 系统管理模块：包括用户权限管理、数据备份与恢复、系统设置等。
